# 工业视觉检测结果复核与质量闭环管理系统

## 1. 项目简介

本项目是一个面向工业视觉质检场景的质量流程管理系统原型，主要用于模拟“视觉检测结果进入企业质量管理流程之后”的业务闭环。

项目以工业表面缺陷检测为背景，围绕生产批次、检测任务、模型检测结果导入、人工复核、不合格记录 NCR、CAPA 整改闭环和质量统计看板展开。

本项目不是完整 MES/QMS 系统，也不是工业现场上线系统，而是一个用于学习和求职展示的后端 / 全栈项目原型。

## 2. 项目目标

本项目的目标不是重新训练模型，而是解决模型检测之后的业务管理问题：

```text
YOLO 模型检测结果
→ 导入系统
→ 生成检测结果记录
→ 人工复核
→ 确认缺陷 / 标记误检 / 需复检
→ 生成 NCR 不合格记录
→ 创建 CAPA 整改闭环
→ Dashboard 展示质量统计
